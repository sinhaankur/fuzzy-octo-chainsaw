Stock Global Intelligence

How It Works

Select any stock in the Stock Monitor panel and the AI instantly analyzes how global events affect it. The system aggregates live news, geopolitical intelligence, market signals, supply chain risks, macro indicators—then uses browser-based LLM to generate actionable insights specific to that stock and sector.

Data Sources Analyzed

📰 Live News Headlines

Sector-relevant news filtered from 30+ global sources

🔍 Intelligence Events

Geopolitical, conflict, and threat alerts from GDELT

📈 Market Signals

Volatility spikes, sector rotations, unusual volume

🔗 Supply Chain Risks

Country exposure and related supply chain threats

⚠️ Geopolitical Threats

Sanctions, conflicts, policy changes affecting sectors

💹 Macro Indicators

VIX, Gold, DXY, Yield Curve, commodity prices

Analysis Workflow

1
User selects a stock from the Stock Monitor portfolio
2
System aggregates all relevant global data for that stock's sector and geographic exposure
3
Browser-based LLM (ONNX Transformers.js) analyzes context and generates insights
4
Results display with confidence score and supporting data points
5
Results are cached to prevent re-analysis on re-selection

Technology Stack

Features

Core Capabilities

  • ✅ 86 Intelligence Panels - Markets, conflicts, supply chain, climate, cyber, maritime, aviation
  • ✅ 30+ Data Sources - GDELT, Reuters, Bloomberg, OpenSky, Sentinel-5P, and more
  • ✅ Real-Time Map - Interactive globe with live events, military movements, and alerts
  • ✅ Portfolio Management - Upload CSV, track country exposure, insider transactions
  • ✅ Local LLM Models - Ollama and LM Studio integration for continued analysis
  • ✅ Backtesting - Test trading strategies against historical data
  • ✅ Playback & Recording - Replay historical market moments
  • ✅ Custom Monitors - Alert on specific tickers, countries, or events
  • ✅ Dark Mode - Easy on the eyes for extended use
  • ✅ 100% Open Source - MIT license, self-hostable

Getting Started

Live Demo

Visit the live application at sinhaankur.github.io/Stockmonitor

Self-Host

Clone the repository and run locally:

git clone https://github.com/sinhaankur/Stockmonitor
cd Stockmonitor
npm install
npm run dev

Usage

  1. Click on Stock Monitor panel or search for a ticker
  2. Select any stock from your portfolio
  3. The Stock Global Intelligence panel instantly generates insights
  4. View the Global Situation panel for context on major events
  5. Optional: Connect local LLM via Settings ⚡ Connect AI button
